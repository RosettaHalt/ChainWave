# ChainWave

波紋を球にぶつけて連鎖させるゲームです。

- [Chain Wave](http://rosettahalt.github.com/ChainWave/)

## ルール

クリックすると波紋が広がります。

波紋と球がぶつかると球が爆発しまた波紋が発生します。

こうして連鎖させていくゲームです。

波紋は一度のみ出すことが出来るのでハイスコアが出る場所をうまく探しましょう。
